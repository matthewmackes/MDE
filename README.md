# MACKES AUDIO PLATFORM

![MACKESAUDIOPLATFORM](MACKESAUDIOPLATFORM.png)

Welcome to the MACKES Audio Platform (MAP) — a source-available suite of VST3 plugins designed for educational testing and validation of larger VST host/loader platforms.

This collection spans a broad range of effects, including:

Classic modulation (chorus, phaser, flanger)

Time-based processing (delay, reverb, echo)

Detailed amplifier simulations (for analysis, training, and verification)

These plugins are intentionally built as “dummy loads”: they are meant to exercise host capabilities (discovery, parameter automation, state save/restore, performance, stability, routing, etc.). They are not intended for production use.
